Nova OS — bundled default DARK wallpaper
========================================

Drop ONE image file in THIS folder to make it the default Dark-mode
wallpaper ("Bloom Dark") at full original quality. Same rules as the light
folder (../wallpapers/):

  • Accepted formats: .png  .jpg  .jpeg  .webp  .avif
  • Recommended size:  2560 x 1600 (or larger).
  • Any filename works — Vite auto-discovers whatever is in this folder.
  • First file (alphabetical) wins if more than one is present.

Bundled asset = ships at original resolution, NO recompression.

After adding/replacing the image, restart `npm run dev`, then switch to Dark
mode to see it.

If this folder has no image yet, "Bloom Dark" falls back to the Mesh wallpaper
so nothing breaks in the meantime.
